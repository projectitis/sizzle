GeneratedStroke

GeneratedStroke class

The result of generating a stroke: the centreline it was grown from, the two offset edges, and the fillable outline that wraps them.

Exposing leftEdge / rightEdge alongside the spine lets callers (and tests) reason about the line the stroke rides — e.g. to check that each edge stays on its own side of the centreline.
