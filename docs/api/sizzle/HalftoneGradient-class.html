offset → Offset
Phase offset in pixels, shifting the dot lattice in the fill's local pixel space (x and y). Since a bake auto-fits its output and re-anchors the lattice to the crop origin, this is how you align the lattices of two independently-baked slices across a shared seam: compute each slice's offset from its position in world space. Offset.zero leaves the lattice at its default (crop-anchored) position.
