capabilities property

Map<String, bool> get capabilities

The capabilities used to flatten the current snapshot. Assigning a new value re-flattens in place and fires onChange. Assigning a value that is equivalent to the current one (treating missing keys as false) is a no-op.

Implementation

Map<String, bool> get capabilities => _capabilities;
set capabilities (Map<String, bool> value)

Implementation

set capabilities(Map<String, bool> value) {
  if (_capsEqual(_capabilities, value)) return;
  _capabilities = value;
  _flatten();
  onChange?.call();
}
