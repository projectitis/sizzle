Load and parse a JX config file from the asset bundle via Services.files. The file is not added to the file cache.

capabilities is a map keyed by modifier name (no # prefix). Any modifier whose name is not present, or whose value is false, is treated as inactive.

static Future<Config> load(
  String path, [
  Map<String, bool> capabilities = const {},
]) async {
  final src = await Services.files.loadString(path: path, cache: false);
  return Config.parse(src, capabilities);
