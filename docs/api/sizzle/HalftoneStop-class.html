HalftoneStop

HalftoneStop class

A single stop on the halftone amount curve.

position is the fractional distance along the gradient axis (0..1) and amount is the dot coverage at that point (0 = no dot / background, 1 = fully solid foreground). Values between stops are linearly interpolated; outside the first/last stop the curve is held flat.

See docs/halftone.md.
