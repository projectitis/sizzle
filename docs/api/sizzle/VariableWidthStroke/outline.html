Builds a fillable outline around spine.

halfWidths gives the half-thickness of the stroke at each spine point (so the full width there is 2 * halfWidths[i]). It must have the same length as spine.

The outline walks up one side of the spine and back down the other, offsetting each point along the local perpendicular, then closes into a single fillable polygon.
