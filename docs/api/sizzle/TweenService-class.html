TweenService

TweenService class

Drives a list of Tweens every frame. The default instance lives at Services.tween and is ticked automatically by SizzleGame.update.

Typical use:

final tween = Services.tween.add(
  from: 0,
  to: 100,
  duration: 1.0,
  ease: Easing.cubicEaseInOut,
  onUpdate: (v) => sprite.position.x = v,
  onComplete: (v) => print('done at $v'),
);
// Later, if needed:
tween.cancel();
